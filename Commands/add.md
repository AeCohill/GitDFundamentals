# git add

WHen changes have been made inside a git repository that you want to keep, you'll need to use the 'git add .' command.

items that have not been "tracked" before will start to be tracked and all other changes will be marked as modified.

Generally, you can get awaywith the command 'get add .' which will add All the files ND DIRECTORIES THAT WERE ADDED, CHANGED, OR DELETED.

Youcan track changes to spific files using "git add file1 file2" which will track spifically file1 or file2.

you can specify tracking directories with 'git add DirectoryName'.
