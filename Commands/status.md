# git status

the command 'git status' when it comes to the, you guessed it status of your repository.

Using 'git status' will show you what changes are being tracked, which are not being tracked, what commits are/need to be pushed/pulled, etc. it is exactly what is sounds like a status check.

itss a command thart is safe ti use at any time. If you want to see whhats going on its good to use 'git status' inbetween commands until you get to know the flow better.

```

git add .
git status

git commit -m "modified language page"
git status

git push
git status
```
